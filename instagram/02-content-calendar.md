# 30-Day Launch Calendar

**Rhythm:** 4 feed posts/week (Tue, Wed, Thu, + one Mon or Fri) + daily Stories. Post by ~11:30am or ~5:30pm. Weekends = Stories only.
**Mix target over the month:** ~60% Reels, ~25% carousels, ~15% static. Pillars rotate A (attention) / B (pain & proof) / C (what we do).

Legend: 🎬 Reel · 🖼️ Carousel · 🟦 Static · 📲 Stories

## After the first month
- **Double down on whatever over-performed.** If Household Name pops, run it weekly and abstract new brands each time.
- **Repurpose everything:** each winning reel → a carousel + 3 Stories + a static quote. One idea, a week of posts.
- **Boost your top 2–3 organic reels** with a small budget to accelerate recognition (amplifying proven content, not buying leads).
- **Review the metrics that matter** (non-follower reach, shares, saves, DM/checklist requests, reviews booked) and cut what isn't landing.

## Batch-production tip
Film all of a month's reels in **1–2 sessions**. Same shirt/setup, knock out 6–8 talking-head reels back to back. The calendar only works if you're not filming from scratch every week.
